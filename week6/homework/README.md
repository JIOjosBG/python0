# Задачи

- триъгълник
- квадрат
- петоъгълник
- шестоъгълник
- пентаграм

2. Напишете програма, която пресмята сумата на четните числа от 1 до N.

3. Напишете програма, която пресмята произведението на нечетните числа от 1 до N.

4. Напишете програма, която изпечатват селдната фигурa:

